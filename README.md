💰 Control de Gastos Semanales

Una aplicación web para registrar y administrar gastos semanales de forma sencilla. Permite agregar gastos y visualizar el presupuesto disponible para ayudar a llevar un mejor control financiero.

🚀 Demo

🔗 https://gastos-semanal-dk.netlify.app/

✨ Características

- Registrar nuevos gastos.
- Mostrar el presupuesto restante.
- Calcular automáticamente el dinero disponible.
- Eliminar gastos.
- Actualización inmediata de los datos.
- Interfaz responsiva.

🛠️ Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript

📂 Instalación

Clona el repositorio:

```bash
git clone https://github.com/Dianakoch/adminPresupuestos.git
```

Entra a la carpeta:

```bash
cd adminPresupuestos
```
